# SQL-Employee-Data-Analysis
Workforce analytics using SQL covering employee distribution, salary analysis, department metrics and business driven data queries.


SQL Employee Data Analysis
Project Overview

This project demonstrates SQL skills applied to HR workforce data to generate insights on employee distribution, compensation trends, and organizational structure. The analysis focuses on solving real business questions using structured queries.

Business Problem

HR teams require workforce insights to support decisions related to hiring, compensation, and organizational planning. Raw employee data alone does not provide clear answers without structured analysis.

This project analyzes employee data to answer key questions:

How many employees exist across departments?
Which departments have the highest salary costs?
What are the hiring trends?
Which roles have the highest compensation?
How is workforce distributed across locations?
